# ジョン（JOHN）

「**ジョン（JOHN）**」は、将棋やシミュレーションRPGのようなルールを取り入れた、トランプを使って2人で遊ぶボードゲームです。

##  特徴

- 4×4マスの小さなボードで、かんたんだけど深いかけひきが楽しめます
- トランプを使って、作戦・読み合い・運のドキドキを味わえます
- 小学生から大人まで、だれでも遊べます
